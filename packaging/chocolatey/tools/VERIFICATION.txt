VERIFICATION

token-usage is distributed on a public Chocolatey feed hosted on this repo's
GitHub Pages. The package downloads its payload from a public GitHub release:

  https://github.com/FactusConsulting/token-usage/releases/

The sha256 checksum embedded in chocolateyinstall.ps1 (__CHECKSUM__ at build
time) is computed by the release workflow against the same source tarball
attached to the GitHub release. To verify manually:

  1. Download token-usage-<version>.zip from the release (no auth needed).
  2. Run: Get-FileHash -Algorithm SHA256 .\token-usage-<version>.zip
  3. Compare against the value baked into chocolateyinstall.ps1.

This package contains NO binaries beyond the install scripts; the payload is
plain Python / PowerShell source from the repo.
